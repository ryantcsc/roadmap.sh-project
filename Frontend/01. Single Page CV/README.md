# Single Page CV - Achmad Fahryan Nur

This project is a simple, single-page CV (Curriculum Vitae) or resume webpage for Achmad Fahryan Nur, a Junior Frontend Developer.

## Description

This webpage is designed to showcase a professional CV on a single HTML page. It includes the following sections:

- **Header:** Displays the name, job title, and contact information.
- **Skills:** Lists key frontend development and design skills.
- **Education:** Placeholder for educational background details.
- **Experience:** Placeholder for previous job roles, achievements, and skills used.
- **Across the Internet:** Suggested links to professional profiles such as LinkedIn and GitHub.

The webpage uses basic HTML5 and inline CSS for styling. The layout is simple and easy to read.

## How to view

Open the `index.html` file in any modern web browser to view the CV webpage.

## Project URL

If hosted, the project can be viewed at: https://roadmap.sh/projects/single-page-cv

Otherwise, open `index.html` locally in your browser to view the CV.
