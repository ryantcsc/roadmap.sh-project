# Changelog Component

Proyek ini adalah implementasi komponen Changelog Timeline sebagai bagian dari latihan dari roadmap.sh.

Tujuan komponen ini adalah menampilkan daftar perubahan atau update dalam bentuk timeline yang rapi dan responsif, sehingga dapat digunakan pada website atau dashboard.

Fitur
• Timeline kiri dan kanan
• Penulisan tanggal menggunakan elemen <time>
• Layout rapi dan mudah dikembangkan
• Cocok untuk latihan frontend pemula maupun menengah

Sumber Proyek

Proyek ini berasal dari roadmap.sh
Link resmi: https://roadmap.sh/projects/changelog-component
